💡 این پروژه چیست؟

Intraverse dApp یک اپلیکیشن غیرمتمرکز (Decentralized Application) است که روی بلاکچین Somnia ساخته شده. این برنامه به کاربران اجازه می‌دهد:

  • مینت (Mint): ساخت توکن‌های جدید به نام "Moon Pass"
  • مدیریت موجودی: مشاهده تمام توکن‌های خود در ۱۳ سطح نادری
  • ارتقا (Upgrade): سوزاندن ۲ توکن از یک سطح برای دریافت ۱ توکن از سطح بالاتر
  • آنالیتیکس: مشاهده آمار تراکنش‌ها، گس مصرفی و نمودارهای روزانه

⛓️ بلاکچین به زبان ساده

بلاکچین مثل یک دفتر حساب عمومی است که هیچ‌کس نمی‌تواند آن را تغییر دهد. وقتی شما یک توکن مینت می‌کنید، این عمل به صورت دائمی در بلاکچین ثبت می‌شود.

اصطلاحات کلیدی:

  • کیف پول (Wallet): مثل حساب بانکی شما در بلاکچین. MetaMask یکی از محبوب‌ترین کیف پول‌هاست.
  • قرارداد هوشمند (Smart Contract): برنامه‌ای که روی بلاکچین اجرا می‌شود و قوانین بازی را مشخص می‌کند.
  • تراکنش (Transaction): هر عملی که روی بلاکچین انجام می‌دهید (مثل مینت کردن).
  • گس (Gas): هزینه‌ای که برای انجام تراکنش پرداخت می‌کنید (مثل کارمزد بانکی).
  • توکن (Token): یک دارایی دیجیتال که مالکیت آن در بلاکچین ثبت شده.

🎭 مینت (Mint) یعنی چه؟

مینت کردن یعنی ساختن یک توکن جدید روی بلاکچین. وقتی شما دکمه "Mint" را می‌زنید:

کیف پول MetaMask باز می‌شود و از شما تأیید می‌خواهد.

یک تراکنش به بلاکچین Somnia ارسال می‌شود.

بلاکچین تراکنش را پردازش می‌کند (چند ثانیه طول می‌کشد).

یک توکن "Moon Pass" جدید با نادری تصادفی (Common تا Supreme) به کیف پول شما اضافه می‌شود.

⚠️ توجه: هر مینت نیاز به مقداری ارز SOMI (ارز بومی شبکه Somnia) برای پرداخت گس دارد. اگر موجودی SOMI ندارید، تراکنش با خطا مواجه می‌شود.

⬆️ ارتقا (Upgrade) چیست؟

سیستم ارتقا به شما اجازه می‌دهد توکن‌های سطح پایین‌تر را به توکن‌های نادرتر تبدیل کنید:

2x Common ──→ 1x Uncommon
2x Uncommon ──→ 1x Rare
2x Rare ──→ 1x Epic
... و به همین ترتیب ...
2x Omega ──→ 1x Supreme 👑

یعنی ۲ توکن از یک سطح سوزانده (burn) می‌شوند و ۱ توکن از سطح بالاتر دریافت می‌کنید. توکن‌های سوزانده شده برای همیشه از بین می‌روند.

📍 توکن‌های ما کجا هستند؟

✅ پاسخ: قرارداد هوشمند این پروژه فقط روی مین‌نت (Mainnet) دیپلوی شده.

آدرس قرارداد: 0xC82E0CE02623972330164657e8C3e568d8f351FA

این قرارداد روی تست‌نت وجود ندارد. بنابراین تمام توکن‌های Moon Pass روی شبکه اصلی Somnia (Chain ID: 5031) هستند.

⚠️ نکته مهم: اگر در MetaMask شبکه تست‌نت را انتخاب کنید، اپلیکیشن نمی‌تواند توکن‌های شما را پیدا کند چون قرارداد در آن شبکه وجود ندارد. حتماً شبکه Somnia Mainnet را انتخاب کنید.

🛠️ نحوه راه‌اندازی و استفاده

پیش‌نیازها

  • مرورگر Chrome یا Firefox
  • افزونه MetaMask نصب شده
  • مقداری ارز SOMI در کیف پول (برای پرداخت گس)

اضافه کردن شبکه Somnia به MetaMask

MetaMask را باز کنید و روی منوی شبکه‌ها (بالای صفحه) کلیک کنید.

روی "Add network" و سپس "Add a network manually" کلیک کنید.

اطلاعات زیر را وارد کنید:

روی Save کلیک کنید. حالا شبکه Somnia به لیست شبکه‌های شما اضافه شده.

استفاده از اپلیکیشن

به آدرس http://localhost:3000 بروید.

روی "Connect Wallet" کلیک کنید و کیف پول MetaMask خود را وصل کنید.

اگر روی شبکه اشتباه هستید، یک بنر زرد ظاهر می‌شود. روی "Switch to Somnia" کلیک کنید.

برای مینت کردن، روی دکمه "Mint" کلیک کنید و تراکنش را در MetaMask تأیید کنید.

پس از تأیید تراکنش، توکن جدید در بخش "Your Inventory" نمایش داده می‌شود.

اجرای پروژه برای توسعه‌دهندگان

سریع‌ترین راه: Docker Compose

# Start all services with one command docker compose up --build # Frontend: http://localhost:3000 # Backend: http://localhost:4000

یا اگر می‌خواهید به صورت دستی اجرا کنید:

# Start PostgreSQL database docker compose up db -d # Push database schema npm run db:push -w packages/backend # Start backend (port 4000) npm run dev:backend # Start frontend (port 3000) — in another terminal npm run dev:frontend

فایل‌های تنظیمات پروژه

برای راه‌اندازی سریع، فایل‌های .env.example در هر پکیج قرار دارند. کافی است آن‌ها را کپی کرده و مقادیر را تنظیم کنید:

cp packages/frontend/.env.example packages/frontend/.env.local cp packages/backend/.env.example packages/backend/.env

همچنین فایل README.md در ریشه پروژه شامل توضیحات کامل درباره تکنولوژی‌ها، معماری و نحوه راه‌اندازی است.

نکته فنی: ایندکسر (Indexer) چیست؟

ایندکسر بخشی از بک‌اند است که به طور مداوم بلاک‌های جدید بلاکچین را بررسی می‌کند. هر تراکنشی که مربوط به قرارداد Moon Pass باشد را پیدا کرده و در دیتابیس ذخیره می‌کند. این داده‌ها برای نمایش آمار و نمودارها در صفحه Analytics استفاده می‌شوند.

🧪 راهنمای تست بدون گس

برای انجام تراکنش روی بلاکچین (مینت، ارتقا) نیاز به ارز بومی شبکه برای پرداخت گس (Gas) دارید. اگر گس ندارید، نگران نباشید! چندین راه برای تست اپلیکیشن وجود دارد:

① بخش‌هایی که بدون گس کار می‌کنند

حتی بدون هیچ ارزی، می‌توانید بخش‌های زیادی از اپلیکیشن را تست کنید:

  • اتصال کیف پول: روی "Connect Wallet" کلیک کنید و MetaMask را وصل کنید — نیاز به گس ندارد
  • تغییر شبکه: بنر "Switch to Somnia" را تست کنید — رایگان است
  • مشاهده موجودی: اگر قبلاً توکن دارید، لیست توکن‌ها نمایش داده می‌شود — فقط خواندن از بلاکچین است و گس ندارد
  • صفحه آنالیتیکس: نمودارها، جدول تراکنش‌ها و کارت‌های آماری — داده از بک‌اند خوانده می‌شود، نه از بلاکچین
  • رابط کاربری: تمام دکمه‌ها، دیالوگ‌ها و صفحات قابل مشاهده و تست هستند

② دریافت SOMI رایگان (فاست مین‌نت)

برای دریافت مقدار کمی SOMI رایگان جهت تست روی مین‌نت، از فاست (Faucet) استفاده کنید:

به سایت Stakely Somnia Faucet بروید.

آدرس کیف پول MetaMask خود را وارد کنید و کپچا را حل کنید.

یک توییت با شناسه درخواست خود بنویسید (سایت لینک و متن را آماده می‌کند).

پس از تأیید توییت، مقداری SOMI به کیف پول شما ارسال می‌شود. این مقدار برای چندین تراکنش مینت و ارتقا کافی است.

③ تست روی تست‌نت (Testnet) با توکن رایگان

اگر نمی‌خواهید با مین‌نت کار کنید، می‌توانید از تست‌نت Somnia استفاده کنید. توکن‌های تست‌نت (STT) کاملاً رایگان هستند.

⚠️ توجه مهم: قرارداد Moon Pass فعلاً فقط روی مین‌نت دیپلوی شده. برای تست کامل روی تست‌نت، باید قرارداد را مجدداً روی تست‌نت دیپلوی کنید. اما می‌توانید از تست‌نت برای تست اتصال کیف پول و رابط کاربری استفاده کنید.

دریافت STT رایگان:

فاست (Faucet) آدرس توضیح
فاست رسمی Somnia testnet.somnia.network فاست رسمی شبکه تست‌نت — ساده‌ترین راه
Gas.zip gas.zip/faucet/somnia تا ۰.۵ STT هر ۲۴ ساعت — بدون نیاز به توییت
دیسکورد Somnia discord.gg/somnia در کانال #dev-chat از تیم توسعه درخواست STT کنید

فعال‌سازی تست‌نت در اپلیکیشن:

# فایل packages/frontend/.env.local را ویرایش کنید: NEXT_PUBLIC_USE_TESTNET=true NEXT_PUBLIC_API_URL=http://localhost:4000 # سپس سرور فرانت‌اند را ری‌استارت کنید

④ تست فقط رابط کاربری (بدون بلاکچین)

حتی بدون اتصال کیف پول می‌توانید بخش‌های مختلف را ببینید:

  • صفحه اصلی (/): دکمه "Connect wallet to mint" نمایش داده می‌شود
  • صفحه آنالیتیکس (/analytics): اگر بک‌اند در حال اجراست، داده‌های واقعی از بلاکچین نمایش داده می‌شود (بدون نیاز به کیف پول)
  • هدر و ناوبری: منوی بالا با لینک‌های صفحات کار می‌کند
  • دیالوگ ارتقا: اگر کیف پول وصل باشد ولی توکن نداشته باشید، دکمه Upgrade غیرفعال نمایش داده می‌شود
